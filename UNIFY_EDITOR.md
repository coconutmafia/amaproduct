## Что расходится (это и есть работа)
| | Пост | Карусель | Сторис |
|---|---|---|---|
| Оболочка | попап `PostImage` | попап `CarouselSlides` | страница `/stories` |
| Порядок | текст→фото→формат | генерит слайды при открытии (до фото) | фото→сценарий→генерация |
| Модель | 1 заголовок+фото | `{cover,slides[],last}` | серия `Frame[]` |
| Выход | 1 картинка | N слайдов + ZIP | N кадров + ZIP, сохр. сеты |
| Точки входа | чат, контент-план | чат, контент-план | страница /stories, контент-план |

## Целевая архитектура
`components/content/ContentEditor.tsx` — единый шелл:
- **Табы формата:** Пост | Карусель | Сторис.
- **Единый базовый сценарий (консистентный порядок):** текст/сценарий → фото (загрузить / выбрать
  загруженное / AI) → опции формата → «Сгенерировать» → превью → правки → скачать/в план.
- **Пер-формат адаптеры** (внутри шелла) инкапсулируют различия: как текст→структура, как маппятся
  фото, что показывает превью, как рендерится. Переиспользуют существующие API (выше).
- Дом: новый маршрут `app/(dashboard)/projects/[id]/create/page.tsx`. Контент-план/чат позже
  диплинкуют сюда с префиллом `?format=&text=`.
